# Проект: Арихтектор

Русский (кратко)

- Цель: практическая попытка реализовать модель, имитирующую человеческий "характер" на основе эмоций.
- Ключевые файлы: neural_network.py, plotting.py, functions.py, inference.py, requirements.txt.
- Входные данные: токенизированные диалоги / метки эмоций. Результат: предсказания классов / поведенческие параметры.
- Быстрый запуск:
  1) Установить зависимости: pip install -r requirements.txt
  2) Запустить обучение/инференс: python neural_network.py
- Примечания: использовать Python 3.8+ и PyTorch; проверьте наличие файлов моделей (pokrov_model.pt, logs_epoch_phase_metrics.pt).

English (brief)

- Purpose: practical attempt to implement a model simulating a human-like "character" driven by emotions.
- Key files: neural_network.py, plotting.py, functions.py, inference.py, requirements.txt.
- Data: tokenized dialogues / emotion labels. Output: class predictions and behavior-related parameters.
- Quick start:
  1) Install deps: pip install -r requirements.txt
  2) Run training/inference: python neural_network.py
- Notes: Requires Python 3.8+ and PyTorch; check for model artifacts (pokrov_model.pt, logs_epoch_phase_metrics.pt).
