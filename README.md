# Нейронная сеть для симуляции "характера", схожего с человеческим

Этот проект представляет собой практическую попытку реализации нейронной сети, способной моделировать черты "характера", схожего с человеческим, на основе эмоций и их соответствующих секторов. Цель проекта — исследовать, как машинное обучение может моделировать и обрабатывать человеческие эмоции для создания моделей поведения, напоминающих человеческие.

## Обзор

Проект использует набор данных эмпатических диалогов для обучения нейронной сети, которая сопоставляет эмоции с заранее определенными "секторами характера" (например, меланхоличный, холерический, флегматичный, сангвинический). Нейронная сеть включает пользовательские функции активации (`FlexiblePReLU`) и весовые коэффициенты, основанные на эмоциях, для симуляции влияния эмоций на принятие решений и поведение.

### Основные особенности:
- **Сопоставление эмоций**: Эмоции сопоставляются с секторами характера, каждый из которых имеет определенные веса, влияющие на поведение нейронной сети.
- **Пользовательская функция активации**: `FlexiblePReLU` динамически регулирует активацию на основе меток эмоций и заранее определенных весов.
- **Набор данных эмпатических диалогов**: Модель обучается на реальных данных диалогов для понимания эмоционального контекста.
- **Многослойная нейронная сеть**: Глубокая архитектура нейронной сети с 20 слоями для моделирования сложного поведения.
- **Визуализация**: Метрики обучения, такие как потери и точность, визуализируются для анализа.

## Структура проекта

### Файлы:
- **`neural_network.py`**: Содержит реализацию нейронной сети, пользовательской функции активации и цикла обучения.
- **`plotting.py`**: Обрабатывает визуализацию метрик обучения и результатов.
- **`requirements.txt`**: Список зависимостей, необходимых для проекта.
- **`README.md`**: Документация проекта.

### Основные компоненты:
1. **FlexiblePReLU**: Пользовательская функция активации, которая регулирует выходные данные на основе меток эмоций и весов.
2. **NeuralNetwork**: Основная архитектура модели, включая слои встраивания, скрытые слои и генерацию параметров, основанных на эмоциях.
3. **Набор данных**: Набор данных `Estwld/empathetic_dialogues_llm` используется для обучения, токенизируется с помощью BERT.
4. **Цикл обучения**: Реализует расчет потерь, обратное распространение ошибки и оптимизацию с использованием Adam.
5. **Визуализация**: Метрики обучения, такие как потери и точность, визуализируются для анализа.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/crie123/llm-br.git
   cd llm-br
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите скрипт обучения:
   ```bash
   python neural_network.py
   ```

## Как это работает

1. **Сопоставление эмоций**: Каждая эмоция сопоставляется с сектором характера (например, "страх" → "меланхоличный").
2. **Пользовательская активация**: `FlexiblePReLU` регулирует выходные данные нейронов на основе весов эмоций.
3. **Обучение**: Нейронная сеть обучается на токенизированных данных диалогов, с расчетом весов классов для сбалансированного обучения.
4. **Визуализация**: Потери и точность обучения визуализируются для мониторинга прогресса.
